# Graduation Project (Front End)

[Live-Demo](https://sky-shifters.vercel.app/)

## OverView
Taier is a modern travel booking platform designed to streamline the flight discovery and reservation process. Developed as a graduation project at Zagazig University, it integrates real-time flight data with an AI-powered travel assistant to provide an end-to-end booking experience.

## Key Features
- **Real-time Flight Search**: Integration with the Amadeus API to fetch live flight offers, fares, and schedules.
- **Advanced Data Filtering**: Dynamic search results with filtering options and itinerary grouping for better user decision-making.
- **AI Travel Assistant**: A built-in chatbot interface that helps users navigate the platform and provides travel insights via a custom backend model.
- **Secure Payments**: Integrated checkout process using Stripe to handle flight bookings safely.
- **User Management**: Personalized user profiles, account settings, and a real-time notification system.
- **Responsive Design**: A fully adaptive UI built to work seamlessly across mobile, tablet, and desktop devices.
- **Light/Dark Mode**

## Tech Stack
- **React** - Javascript framework
- **Vanilla css** - styling
- **Context api** - state management
- **Vercel** - Deployment
